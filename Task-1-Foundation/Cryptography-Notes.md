# Cryptography Basics

## Encryption

Encryption converts readable data (Plaintext) into unreadable data (Ciphertext).

## Types of Encryption

### Symmetric Encryption
Uses the same key for encryption and decryption.

Examples:
- AES
- DES

### Asymmetric Encryption
Uses two keys:
- Public Key
- Private Key

Examples:
- RSA

## Hashing

Hashing converts data into a fixed-length value.

Examples:
- MD5
- SHA-256

## SSL/TLS

SSL/TLS provides secure communication between client and server over the internet.
