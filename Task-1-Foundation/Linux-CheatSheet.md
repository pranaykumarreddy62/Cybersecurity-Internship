# Linux Cheat Sheet

## Basic Commands

### pwd
Current directory

```bash
pwd
```

### ls
Files మరియు folders 

```bash
ls
```

### cd
Folder change

```bash
cd Documents
```

### mkdir
కొత్త folder create 

```bash
mkdir test
```

### rmdir
Folder delete 

```bash
rmdir test
```

### touch
కొత్త file create

```bash
touch file.txt
```

### cat
File content 

```bash
cat file.txt
```

### chmod
File permissions 

```bash
chmod 777 file.txt
```

### ip a
IP Address 

```bash
ip a
```

### ping
Network check

```bash
ping google.com
```
